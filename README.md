# Proyectos destacados

## 🏥 Infraestructura de telemedicina y conectividad - 2023 (Académico)

Proyecto enfocado en la reducción de brechas de acceso a servicios de salud mediante una arquitectura de red empresarial escalable, segura y de alta disponibilidad basada en Cisco Enterprise Architecture.

### Implementaciones realizadas

- Diseño e implementación de red corporativa basada en Cisco Enterprise Architecture
- Segmentación de red mediante VLANs y switching multicapa
- Configuración de EtherChannel, troncales y alta disponibilidad
- Implementación de routing dinámico OSPF y BGP
- Implementación de DMVPN e IPsec Site-to-Site VPN
- Configuración de VPN Clientless para acceso remoto seguro
- Implementación de servicios DHCP, DNS, NTP y SMB
- Despliegue de servidores Ubuntu Server para servicios corporativos
- Implementación de Apache2 y Nginx
- Instalación y configuración de Jitsi Meet para telemedicina
- Implementación de OpenEMR para registros médicos
- Integración entre OpenEMR y Jitsi Meet
- Aplicación de medidas de seguridad lógica y segmentación
- Desarrollo de documentación técnica, análisis de riesgos y planes de mantenimiento
- Gestión de proyecto bajo metodología PMBOK

## 🖥️ HomeLab de virtualización y servicios self-hosted

Laboratorio personal basado en Proxmox VE orientado a virtualización, networking y operación de servicios self-hosted.

El entorno está diseñado para experimentar con infraestructura moderna, despliegue de servicios, segmentación lógica y administración de sistemas Linux en escenarios similares a entornos reales.

#### VM Docker
- Uptime Kuma
- Pi-hole (planeado mover a LXC)
- Nextcloud
- Portainer

#### Contenedores LXC
- Nginx Proxy Manager
- Samba / SMB
- Tailscale
- Cloudflared Tunnel
- Grafana (planeado)
- Prometheus (planeado)
- WireGuard (planeado)

### Objetivos del laboratorio

- Simular entornos de infraestructura reales
- Validar despliegues y configuraciones
- Practicar administración Linux y networking
- Implementar monitoreo y observabilidad
- Gestionar persistencia y almacenamiento
- Documentar arquitectura e infraestructura
- Probar servicios de red y acceso remoto
- Fortalecer conocimientos de virtualización y contenedores
